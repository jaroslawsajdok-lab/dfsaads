Zbuduj stronę parafii jawornik.eu jako motyw WordPress (najlepiej child theme dla GeneratePress lub Astra), tak aby po wdrożeniu cała obsługa treści była w panelu WordPress.

WYMAGANIA OGÓLNE

Strona ma być klasyczna w układzie, ale z mocno nowoczesnymi elementami UI (glassmorphism menu, delikatne animacje, świetna typografia).

Kod: PHP + HTML + CSS + minimalny JS (bez ciężkich frameworków).

Wszystko responsywne, szybkie, dostępne (a11y).

Zadbaj o Core Web Vitals (lazy load, optymalizacja wideo, brak wielkich bibliotek).

NAJWAŻNIEJSZE: START = WIDEO Z DRONA (FULLSCREEN)

Na stronie głównej na start pełnoekranowy background video (autoplay, loop, muted, playsinline).

Jeśli autoplay zablokowany: pokaż poster (obraz) + przycisk “Odtwórz”.

Na wideo: tytuł „Parafia Ewangelicka w Wiśle Jaworniku” + podtytuł + strzałka „Przewiń”.

Menu (sticky) ma się pojawić po pierwszym scrollu (albo po chwili) i mieć efekt blur/translucent.

WORDPRESS – EDYTOWALNOŚĆ (bez kodu)
Użyj:

Custom Post Types + Custom Fields (ACF) albo Gutenberg Blocks (preferowane).
Treści mają być zarządzalne w WP:

Aktualności – jako standardowe wpisy (posty) + opcjonalnie wyświetlanie feedu FB (fallback: ręczne wpisy).

Kalendarz – jako CPT „Wydarzenia” z polami: data, godzina, typ, opis, miejsce. Widok: lista najbliższych + mini-kalendarz.

Grupy w parafii – jako CPT „Grupy” (nazwa, opis, prowadzący, termin, ikona).

Nagrania – sekcja z linkami YouTube (CPT „Nagrania” lub pole w ustawieniach strony).

Galeria – CPT „Galerie” + galeria zdjęć (media library).

FAQ – CPT „FAQ” (pytanie/odpowiedź).

Dom Gościnny – edytowalny blok z linkiem do https://osrodek.jawornik.eu

Kontakt – edytowalne pola: adres, tel, mail, godziny + mapa.
